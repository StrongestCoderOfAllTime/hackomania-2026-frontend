# Welcome to your Lovable project

## Description

This is a simple energy simulator that calculates the potential savings from making small changes to your energy usage.

## Installation

1. Clone the repository

```bash
git clone git@github.com:StrongestCoderOfAllTime/hackomania-2026-frontend.git
```

2. Navigate into the repository

```bash
cd hackomania-2026-frontend/
```

3. Install dependencies

```bash
npm install
```

4. Run the development server
```bash
npm run dev
```

## Tech Stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
